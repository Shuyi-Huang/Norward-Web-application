# Norward â€” Company Website

Norward is a clean, responsive company website developed for a local business.  
The project focuses on clear structure, visual consistency, and a user-friendly layout using pure HTML and CSS.

This website was designed and developed as a real-world client project, with attention to maintainability and presentation quality.



## Project Overview

- Purpose: Company introduction and online presence
- Audience: General customers and partners
- Approach: Lightweight, fast-loading static website
- Design focus: Simplicity, readability, and responsive layout



## Features

- Responsive layout for desktop, tablet, and mobile
- Clean and consistent UI design
- Well-structured HTML5 semantic elements
- Reusable and maintainable CSS styling
- Optimised image and asset handling
